# User Manager

--------------
API STRUCTURE
--------------

----------------
Insert a user
----------------
Route-type: Post
----------------
Route: /api/v1/user
----------------
request body: name,username,password
----------------
----------------
Get User Details
----------------
Route-type: get
----------------
Route : /api/v1/user/:id
----------------

----------------
Update an user
---------------
Route-type: update
---------------
Route : /api/v1/user/:id
---------------
request body: name,username
---------------

---------------
Delete a user
---------------
Route-type: Delete
---------------
Route:/api/v1/user/:id
